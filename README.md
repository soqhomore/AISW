# 🐰 SleepBunny - 토끼와 함께하는 수면 도우미

편안한 밤을 위한 귀여운 토끼 친구와 함께하는 웹 기반 수면 도우미 애플리케이션입니다.

![SleepBunny](assets/images/bunny-day.png)

## 💡 프로젝트 개요

### 제작 동기
현대인들은 스트레스와 불안으로 잠들기 어려워하는 경우가 많습니다. 특히 취침 전 스마트폰을 사용하면서도 편안한 수면을 유도할 수 있는 도구의 필요성을 느꼈습니다. 이에 **심리적 안정감**을 주는 귀여운 캐릭터, **수면을 방해하지 않는 다크 모드**, **화이트 노이즈**, **독서 기능**을 결합한 웹 애플리케이션을 개발하게 되었습니다.

### 타겟 사용자
- 😴 수면에 어려움을 겪는 대학생 및 직장인
- 📱 취침 전 스마트폰을 사용하는 사람들
- 🎵 화이트 노이즈나 ASMR로 잠드는 것을 선호하는 사람들
- 📚 잠들기 전 책을 읽으며 휴식을 취하고 싶은 사람들

### 제공 가치
- **심리적 안정감**: 귀여운 토끼 캐릭터와의 상호작용으로 힐링
- **수면 환경 조성**: 다크 모드 + 화이트 노이즈로 최적의 수면 환경 제공
- **재미있는 인터랙션**: 먹이 주기, 독서 등의 통계 기록으로 사용하는 재미
- **접근성**: 백엔드 없이 브라우저만 있으면 언제 어디서나 사용 가능

## ✨ 주요 기능

### 1. 🎉 환영 화면
- 앱 시작 시 랜덤으로 응원 메시지 표시
- 8가지 다른 따뜻한 메시지로 하루의 마무리를 격려합니다

### 2. 🥕 먹이 주기
- 토끼에게 다양한 먹이를 줄 수 있습니다
  - 당근 🥕
  - 양배추 🥬
  - 사과 🍎
  - 상추 🥗
- 먹이를 줄 때마다 귀여운 애니메이션과 함께 토끼가 반응합니다
- **애니메이션 순서**:
  1. "먹고 있어요." (점이 1~3개로 반복, 3초)
  2. "잘 먹었어요! 😊" (2초)
  3. "자고 있어요 💤" (idle 상태 복귀)
- **스마트 조사**: 한글 조사(을/를)가 자동으로 적용됩니다
- 먹이 주기 횟수가 자동으로 기록됩니다

### 3. 📚 독서 기능
- 10편의 엄선된 고전 문학 작품 수록
  - **시(詩)**: 별 헤는 밤, 진달래꽃, 님의 침묵, 서시, 봄은, 눈오는 밤, 광야
  - **소설/단편**: 어린 왕자, 사랑손님과 어머니, 메밀꽃 필 무렵
- 읽은 위치가 자동으로 저장되어 다음에 이어서 읽을 수 있습니다
- 토끼가 함께 책을 읽는 모습을 감상할 수 있습니다

### 4. 🎵 화이트 노이즈
- 4가지 편안한 소리를 제공합니다 (실제 오디오 파일 포함!)
  - 📖 책 넘기는 소리 (1개 파일)
  - 🌧️ 빗소리 (1개 파일)
  - 🌲 숲속 소리 (1개 파일)
  - 💤 백색 소음 (2개 파일 중 랜덤 재생)
- **미지원 항목** (파일 크기 제한):
  - 🌊 파도 소리 (향후 추가 예정)
  - 🔥 모닥불 소리 (향후 추가 예정)
- **새로운 플로우**:
  1. 소리 선택 → 2. 타이머 선택 (10분/30분/1시간/무제한) → 3. 메인 화면에서 재생
- **음악 재생바**: 토끼 이미지 **상단**에 Glassmorphism 스타일로 표시 (겹치지 않음)
  - 현재 재생 중인 소리 표시
  - 남은 시간 표시
  - 재생/일시정지/정지 컨트롤
- **랜덤 재생**: 각 카테고리의 여러 파일 중 하나를 랜덤으로 선택
- **0.5초 딜레이**: 부드러운 재생 시작
- **무한 반복**: 일시정지/정지할 때까지 계속 재생
- 볼륨 조절 가능

### 5. 🌓 다크 모드
- 눈에 편안한 다크 모드 지원
- 취침 전 낮은 밝기로 눈의 피로를 줄입니다
- 설정이 자동으로 저장됩니다

### 6. 👤 프로필 & 통계
- 사용자 이름 설정 (토끼가 이름을 불러줍니다!)
- 상세 활동 통계 확인
  - 먹이 준 횟수 (음식별 통계)
  - 읽은 책 목록
  - 음악 들은 횟수 (소리별 통계)
  - 이름 불러준 횟수
  - 앱 사용 횟수

### 7. 💕 이름 부르기
- 설정한 이름으로 토끼를 불러보세요!
- 랜덤한 감정 이모지와 함께 귀여운 반응을 보여줍니다

## 🚀 시작하기

### 설치 방법

1. 이 저장소를 클론하거나 다운로드합니다
```bash
git clone [repository-url]
cd sleep-app
```

2. 웹 브라우저로 `index.html` 파일을 엽니다
   - 파일 탐색기에서 `index.html`을 더블클릭하거나

```bash

python -m http.server 8000

# Node.js http-server 사용
npx http-server -p 8000
```

그런 다음 브라우저에서 `http://localhost:8000`으로 접속하세요.

## 📱 사용 방법

### 1단계: 앱 시작
- 환영 화면에서 "앱 시작하기" 버튼을 클릭합니다
- 따뜻한 응원 메시지를 받습니다

### 2단계: 토끼와 상호작용
- **먹이 아이콘 (원형) 🥕**: 토끼에게 간식을 주세요
- **책 아이콘 (원형) 📚**: 고전 문학을 읽으며 휴식하세요
- **소리 아이콘 (원형) 🎵**: 편안한 화이트 노이즈를 들으세요

### 3단계: 설정 조정
- 상단의 **전구 아이콘 💡**: 다크 모드 켜기/끄기
- **설정 아이콘 ⚙️**: 볼륨 조절 및 기타 설정
- **프로필 아이콘 👤**: 통계 확인 및 이름 설정

## 🎨 기술 스택 & 디자인

- **HTML5**: 시맨틱 마크업
- **CSS3**: 
  - CSS Variables (테마 지원)
  - **Glassmorphism UI** - 반투명 유리 효과
  - **Backdrop Filter** - 배경 블러 효과
  - Flexbox & Grid 레이아웃
  - CSS Animations
  - 반응형 디자인 (Mobile First)
  - 동적 배경 이미지 (라이트/다크 모드별, 모바일/웹별)
- **JavaScript (Vanilla)**:
  - ES6+ 문법
  - 모듈식 구조
  - Local Storage API
  - Web Audio API
  - Custom Events

- **Glassmorphism**: 투명하고 현대적인 유리 효과 UI
- **동적 배경**: 토끼 캐릭터가 포함된 풀 배경 이미지
- **귀여운 폰트**: Gowun Dodum (고운돋움) - 가독성과 귀여움을 모두 갖춘 폰트
- **원형 토끼**: 다크/라이트 모드에 맞는 토끼 이미지를 원형으로 표시
  - 라이트 모드: bunny-day.png

- **컬러 팔레트**:
  - 라이트 모드: 투명-흰색-분홍색 (#ff9ecd)
  - 다크 모드: 투명-분홍-갈색 (#ffb3d9)
  - 텍스트: 채도 낮은 분홍-갈색 계열
- **부드러운 애니메이션**: 모든 인터랙션에 우아한 전환 효과

### 색상 변경

`css/styles.css` 파일의 `:root` 섹션에서 색상 변수를 수정하세요:

```css
:root {
  --bg-primary: #FFF5F5;      /* 배경색 */
  --accent: #FFB5B5;          /* 강조 색상 */
  /* ... 더 많은 변수들 */
}
```

### 응원 메시지 추가

`js/app.js`의 `welcomeMessages` 배열에 메시지를 추가하세요:

```javascript
welcomeMessages: [
  "오늘 하루도 수고하셨어요!",
  "당신의 메시지 추가 ✨"
]
```

### 문학 작품 추가

`js/books.js`의 `books` 배열에 새로운 작품을 추가하세요:

```javascript
{

  title: '책 제목',
  author: '작가',
  type: 'poem', // or 'novel'
  content: `책 내용...`
}
```

## 📚 학습 내용 및 성찰

### 이 프로젝트를 통해 배운 것
- **프론트엔드 개발**: HTML, CSS, JavaScript의 기본부터 고급 기능까지 실습
- **로컬 스토리지 활용**: 백엔드 없이 데이터를 저장하고 관리하는 방법
- **UX/UI 디자인**: 사용자 경험을 고려한 Glassmorphism 디자인 구현
- **반응형 웹**: 모바일과 데스크톱 환경 모두를 고려한 레이아웃 설계
- **Web Audio API**: 오디오 파일 재생 및 타이머 관리
- **문제 해결**: 한글 조사 처리, 다크모드 이미지 전환 등 실제 개발 중 발생하는 문제 해결

### 어려웠던 점과 해결 방법
1. **한글 조사 처리**: 받침 유무에 따라 '을/를', '이/가'를 자동으로 선택하는 로직 구현
2. **다크모드 이미지 전환**: CSS `content` 속성의 한계를 극복하고 JavaScript로 동적 처리
3. **음악 플레이어 UI**: 토끼 이미지와 겹치지 않으면서도 접근성 좋은 위치 선정
4. **반응형 디자인**: 스크롤 없이 모든 콘텐츠를 표시하기 위한 크기 조정

### 느낀 점
웹 개발이 단순히 코드를 작성하는 것이 아니라, 사용자의 필요와 경험을 고려하여 문제를 해결하는 과정임을 배웠습니다. 특히 수면이라는 민감한 주제에 대해 심리적 안정감, 눈의 피로, 사용 편의성 등 다양한 측면을 고려해야 했고, 이를 통해 개발자로서 사용자 중심 사고의 중요성을 깨달았습니다.
